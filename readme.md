How to install react in Windows?
-
### For Normal React
    npm i -g create-react-app
    npx create-react-app [Your_App_Name]
    cd [Your_App_Name]
    npm start

### For Vite Tool React
    npm create vite@latest
    cd [Your_App_Name]
    npm i
    npm run dev
